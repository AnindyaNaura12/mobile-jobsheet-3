Program Matematika Sederhana (Dart)

Gambaran Program
Program ini adalah program sederhana yang dibuat dengan Dart untuk latihan percabangan if-else dan perulangan for.
Alur program: 
1. Melihat menu
2. Memilih fitur
3. Memasukkan angka
4. Program menampilkan hasil

Fitur program meliputi:
1. Cek Bilangan Prima: Menentukan apakah angka yang dimasukkan termasuk bilangan prima
2. Hitung Faktorial: Menghitung faktorial dari angka yang dimasukkan
3. Validasi Input: Menolak input angka ≤ 0 agar program berjalan dengan benar|

Kegunaan Masing-Masing Branch
Branch:
- main --> Branch utama yang menyimpan final program 
- input-feature --> Branch untuk input user dan menu dasar 
- prime-feature --> Branch untuk logika pengecekan bilangan prima 
- factorial-feature --> Branch untuk perhitungan faktorial
